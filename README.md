🎬 MovieFlix – Movie Search Web App

A modern movie search application built using HTML, CSS, and JavaScript. This app allows users to search for movies, view details, and save favorites with a Netflix-style UI.

🚀 Features
🔍 Search movies dynamically using API
🎥 Display movie posters, titles, and release year
📄 Click on a movie to view full details (popup)
⭐ Add movies to Favorites (stored in localStorage)
📂 View saved favorite movies
🎬 Netflix-style responsive UI with hover effects
⚠️ Error handling (no movies found, invalid search)
